# Chatroom
Chatroom with Django Channels

Bài tập lớn Python

**Mục tiêu:** 
- Tạo ra một Chatroom đơn giản để nhắn tin.
- Luyện tập áp dụng những kiến thức đã học vào một project thực tế.
- Tìm hiểu về Django, Channels (Websocket)

**Chức năng:**
- Tạo phòng
- Tham gia phòng
- Nhắn tin
- Kết bạn
